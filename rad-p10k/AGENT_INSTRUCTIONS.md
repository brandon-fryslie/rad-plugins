# AGENT_INSTRUCTIONS.md — rad-p10k

## Setup

Add these two lines to `~/.rad-plugins`. Order matters — rad-p10k must set
config variables before p10k reads them.

```
brandon-fryslie/rad-plugins rad-p10k
romkatv/powerlevel10k powerlevel10k
```

## Dependencies

- [powerlevel10k](https://github.com/romkatv/powerlevel10k)
- Nerd Font (recommended: JetBrainsMono Nerd Font)
